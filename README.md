# Smart Hospital IoT System

## Overview

This project is a smart hospital infrastructure designed using Cisco Packet Tracer to improve patient safety, automate room controls, and strengthen hospital security through IoT technologies.

The system enables patients to send emergency alerts using smartphones, allows automatic control of windows based on temperature conditions, and provides real-time movement monitoring through sensors and surveillance devices.

Network communication is ensured through DHCP automatic addressing, wireless access points, and OSPF dynamic routing between multiple segments.

This project demonstrates practical skills in networking, IoT integration, automation, and secure infrastructure design.

## Technologies Used

- Security Monitoring
